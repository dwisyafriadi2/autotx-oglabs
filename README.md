# 0G Labs Auto TX
## How to run bot faucet

1. Clone
```
git clone https://github.com/dwisyafriadi2/autotx-oglabs.git
```
2. install package

```
cd autotx-oglabs
```
3. Edit PK
edit your private key
```
nano privatekey.txt
```

. start

```
python3 run.py 
```
or
```
py run.py
```
